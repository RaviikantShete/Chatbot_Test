## Quick Start

### 1. Install
```bash
npm install
npx playwright install chromium
```

### 2. Configure
Ensure `.env` is present with valid configuration.
Contact the author if `.env` is missing or needs updating.

### 3. Run Tests
```bash
npm run test:report
```

### 4. View Report
Open `reports/html/index.html` in your browser.

---

## Test Commands

| Command | Description |
|---------|-------------|
| `npm run test:report` | Run all tests + generate report |
| `npm run test:smoke` | Critical path only |
| `npm run test:regression` | Full regression suite |
| `npm run test:headed` | Run with visible browser |
